# Cypress - Automação de Testes E2E (Swag Labs)

Projeto de automação criado para demonstrar testes end-to-end em uma jornada completa de compra. 

## O que foi automatizado
- Login com sucesso  
- Jornada completa de compra (adicionar produto → carrinho → checkout → finalização)  
- Uso de **Custom Command** (`cy.login()`)  
- Validações de regras de negócio e mensagens  

## Tecnologias
- Cypress 15.12.0  
- JavaScript  
- Custom Commands  
- Vídeos e screenshots automáticos  

## Como rodar
```bash
npm install
npx cypress run
